IV. PERSONALIZATION AND DESIGN FILE REQUIREMENTS
- Accepted File Formats: High-resolution files in PNG, JPG, JPEG, SVG, or PDF format.
- Resolution: Files should have a minimum of 300 DPI. Avoid screenshots or small, pixelated images.
- Color Profile: We use a standard CMYK color profile.
- WhatsApp Files: If sending designs via WhatsApp, please send the file as a document, not a photo, to maintain full quality.
- Tees Print Sizes: A5 Mini (5" x 8") to A2 Max (16" x 23").
- Totes Print Size: A4 Mid (8" x 11").
- File Limits: You can upload at least three files per order, with each file under 10MB.
- Placement: You can request specific design placement (front, center, corner) by adding a note during checkout or uploading a reference image. Clear file naming (e.g., "front tote image.png") is essential.
- Change Window: Once an order is placed and the design file is confirmed, changes cannot be made after a strict **6-HOUR WINDOW**.
- Color Guarantee: We cannot guarantee exact color matches due to variations in monitor settings and material absorption rates.
- **DESIGN UPLOAD GUIDE:** https://iamxis.com.ng/design-upload-guide/

V. PRODUCT CARE AND DURABILITY
- Tees Care (100% breathable cotton): Wash inside out in cold water and hang dry. Avoid using bleach and do not iron directly over the custom print.
- Totes Care (100% durable cotton): Should be spot-cleaned gently with mild detergent and a soft cloth. Avoid machine washing or scrubbing totes.
- Mugs Care (Glossy and Magic): For maximum print longevity, we strongly recommend hand washing them with warm water and mild soap. Avoid using dishwashers for mugs.
- Totes Features: Totes are designed for everyday carry; select totes include a lined interior with pockets for added functionality.
